Step 6: Mechine Learning

After cleaning getting more insight about the data, iit is time to start Mechine learning part. I have tried 5 Different algorithm. I used Python Sklearn library.

Algoritms I have used

  • Knn
  • Random Forest
  • Xgboost
  • SVC

After doing grid search and Hyper parametter Tunning I got the best score with the Random Forest algorithm. 80.70 is the clasification score with Random forest. I got 80.36 with Xgboost, 76.57 with Knn(k=7) and 73.86 with SVC.I used GridsearchCV to find best parametters.

After Hyper parametter Tunningusing GridsearchCV best parametters for Random Forest Model
  • bootstrap = True
  • n_estimators=1000
  • max_depth = 16
  • min_samples_split = 2
  • max_features= 'auto'
  • min_samples_leaf = 1
